# Module 07 — Streamlit Dashboard

A simple web dashboard built with Streamlit, based on the wireframe in this directory.

## Layout

- **Header** — title + primary action button
- **Sidebar** — navigation (Overview, Notes, Tasks, Reports, Settings)
- **Main area** — KPI metric cards + recent items list
- **Footer** — version string

## Setup

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

The app opens at `http://localhost:8501`.
