# Module 07 Diff Notes

## Goal

Create a working UI from a wireframe image using Claude Code.

## Result

Claude generated a Streamlit dashboard based on the provided wireframe.

## What matched the wireframe

- Header area was implemented.
- Main dashboard content was implemented.
- KPI/stat cards were implemented.
- Table/list section was implemented.

## Differences from the wireframe

- Styling is approximate, not pixel-perfect.
- Layout is functional rather than production-polished.
- Colors, spacing, and fonts may differ from the original wireframe.

## Lesson Learned

Claude can create a useful first-pass UI from a screenshot or wireframe, but the output still needs human review for layout, visual quality, accessibility, and product fit.
